Configuration initiale HAKit

Commencez par installer et configurer HAKit dans votre projet.

Installation 
npm install @hakit/core @hakit/components
Configuration de base 
import { HassConnect, ThemeProvider } from '@hakit/core';
import { Row, Column, WeatherCard } from '@hakit/components';

const customTheme = {
  colors: {
    primary: '#1FB8CD',
    secondary: '#FFC185',
    background: '#2C3E50',
    surface: '#34495E',
    text: '#FFFFFF'
  },
  borderRadius: '20px',
  boxShadow: '0 8px 32px rgba(0, 0, 0, 0.2)'
};

function App() {
  return (
    <HassConnect hassUrl="http://your-home-assistant-url:8123">
      <ThemeProvider theme={customTheme}>
        {/* Votre dashboard ici */}
      </ThemeProvider>
    </HassConnect>
  );
}

Structure du Layout

Organisez vos cartes avec un système de grille responsive.

Layout Principal 
<Row style={{ minHeight: '100vh', gap: '0' }}>
  {/* Sidebar */}
  <Column style={{ width: '240px' }}>
    <SidebarCard>
      {/* Navigation */}
    </SidebarCard>
  </Column>
  
  {/* Main Content */}
  <Column style={{ flex: 1, padding: '24px' }}>
    <div style={{
      display: 'grid',
      gridTemplateColumns: 'repeat(3, 1fr)',
      gap: '20px',
      height: 'fit-content'
    }}>
      {/* Cartes ici */}
    </div>
  </Column>
</Row>
Grid CSS Responsive 
.dashboard-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
  padding: 24px;
}

@media (max-width: 1024px) {
  .dashboard-grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (max-width: 768px) {
  .dashboard-grid {
    grid-template-columns: 1fr;
  }
}
